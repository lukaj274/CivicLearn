# CivicLearn

## What is CivicLearn?
CivicLearn is a game in which players can design their own city. It is available on Linux, Mac, and Windows.

## Why?
CivicLearn was made for the LUMA Startathon 2025. It aims to solve problems in the fields of Education and Infrastructure by:

- Providing a fun and educational game for children
- Helping these children build skills that they will be able to use as adults to design cities
